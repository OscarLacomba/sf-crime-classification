# sf-crime-classification
Crime classification using supervised machine learning in San Francisco
# I encourage you to use and practice the entire data science lifecycle, while exploring the dataset visually via Exploratory Data Analysis (EDA).

# Tasks

1.Import the necessary libraries
2.Load the Dataset
3.EDA
4.Split the Dataset
5.Initialize and train at least 3 supervised classification models
6.Make predictions
7.Evaluate the model with metrics suitable for classification
8.Write within your notebook at least 3 conclusions from the analysis
Your main goal is to attain an overall accuracy higher than 80%.
